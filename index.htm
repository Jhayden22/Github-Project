About GitHub

GitHub is a platform for hosting and collaborating on code repositories by using a version control system to track changes made to files over time.

Environment Setup

Download VS code and Configure GitHub account

Configure Git by opening VS Code Terminal and entering:

    git config --global user.name "Your Name"
    git config --global user.email "Your Email"

Creating & Cloning Repositories

To create a new repository on GitHub, go to your GitHub account, click on the "+" icon in the top-right corner, and click "New repository." Enter a name for your repository,description, and choose "public" or "private" for visibility.

Navigate to your repository in the repository menu on the left

In the dropdown menu under URL, ensure that it says "HTTPS".

Copy the URL from the dropdown menu.

Open a new terminal in VS code in file you want to clone. Enter "git clone (paste URL)"

Basic Git Operations

Commit&Push

After making changes to files in your repository, you need to commit those changes. Navigate to "Source Control" in VS Code Menu. click the command dropdown and select "commit and push" to commit and push changes to your local repository.

Creating Branches

Branches allow you to work on different features or versions of your project without interfering with the main code.
when creating a new branch, use the command "git checkout -b new-branch-name",
or in VS code you can quickly change branches by clicking on the current branch on the bottom left
and selecting the branch you would like to upload to.
